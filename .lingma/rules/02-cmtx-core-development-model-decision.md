# @cmtx/core 核心库开发规则 (模型决策)

## 适用场景

当开发涉及以下功能时自动生效：

- Markdown 图片解析和提取
- 图片引用分析
- 图片路径处理
- 正则表达式模式匹配

## 核心功能规范

### Parser 模块

- 使用 remark + rehype 进行 AST 解析
- 支持 Markdown 和 HTML 两种语法
- 提供准确的位置信息（行号、列号）
- 返回标准化的 ParsedImage 接口

### Filter 模块

- 支持多种过滤条件组合
- 提供高效的目录遍历能力
- 支持 glob 模式匹配
- 实现 ignore 规则机制

### Replace 模块

- 支持字段级别的精确替换
- 保持原有语法结构完整性
- 提供详细的替换记录
- 支持批量操作和回滚

### Delete 模块

- 实现安全删除策略
- 支持引用检查机制
- 提供多种删除选项（移动、硬删除等）
- 记录删除操作日志

## API 设计原则

- 异步优先：所有 IO 操作返回 Promise
- 回调支持：提供 logger 等回调参数
- 类型安全：完整的 TypeScript 类型定义
- 错误处理：明确的错误类型和处理机制
