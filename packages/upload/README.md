# @cmtx/upload

Object storage upload helpers for @cmtx/core.

## Status

🚧 **Under Development** - This package is currently in early development.

## Planned Features

- Upload images to Alibaba Cloud OSS
- Automatic reference replacement in Markdown files
- Support for multiple cloud storage providers (AWS S3, Azure Blob, etc.)

## Installation

```bash
npm install @cmtx/upload ali-oss
# or
pnpm add @cmtx/upload ali-oss
```

Note: Cloud storage SDKs (like `ali-oss`) are peer dependencies and must be installed separately.
