# 知乎 Markdown 格式规范

## 1. 标题规范

知乎编辑器支持的标题层级：

- 一级标题：`# 标题`
- 二级标题：`## 标题`
- 三级标题：`### 标题`

知乎平台本身对标题层级没有严格限制，但为了良好的阅读体验，建议：

- 文章主标题使用一级标题 `#`
- 章节标题使用二级标题 `##`
- 小节标题使用三级标题 `###`

## 2. 与标准 Markdown 的差异

### 2.1. 标题层级

标准 Markdown 文档通常使用 `##` 作为文章主标题（遵循 KiloCode 规范），而知乎平台使用 `#` 作为主标题。

| 标准 Markdown | 知乎格式 |
| --- | --- |
| `## 文章标题` | `# 文章标题` |
| `### 章节标题` | `## 章节标题` |
| `#### 小节标题` | `### 小节标题` |

### 2.2. YAML Front Matter

知乎编辑器不支持 YAML Front Matter，需要删除文档开头的元数据块：

```markdown
---
title: 文章标题
author: 作者
date: 2024-01-01
---
```

## 3. 支持的 Markdown 语法

### 3.1. 基础格式

- **粗体**：`**文字**`
- *斜体*：`*文字*`
- ~~删除线~~：`~~文字~~`
- `行内代码`：`` `代码` ``

### 3.2. 代码块

```markdown
```python
def hello():
    print("Hello, World!")
```
```

### 3.3. 列表

无序列表：

```markdown
- 项目一
- 项目二
- 项目三
```

有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 3.4. 链接和图片

链接：

```markdown
[链接文字](https://example.com)
```

图片：

```markdown
![图片描述](https://example.com/image.png)
```

### 3.5. 引用

```markdown
> 这是一段引用文字
```

### 3.6. 分隔线

```markdown
---
```

### 3.7. 表格

```markdown
| 列1 | 列2 | 列3 |
| --- | --- | --- |
| A | B | C |
| D | E | F |
```

## 4. 注意事项

1. **图片上传**：知乎支持直接粘贴图片或拖拽上传，外链图片可能无法正常显示
2. **数学公式**：知乎支持 LaTeX 数学公式，使用 `$` 或 `$$` 包裹
3. **代码高亮**：知乎支持多种编程语言的语法高亮
4. **目录**：知乎会自动根据标题生成文章目录

## 5. 发布流程

1. 使用转换脚本将 Markdown 转换为知乎格式
2. 复制转换后的内容
3. 粘贴到知乎编辑器
4. 检查格式是否正确
5. 上传本地图片（如有）
6. 发布文章
