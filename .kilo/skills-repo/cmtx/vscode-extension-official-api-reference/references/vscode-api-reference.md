## Official URL

VS Code API Reference <https://code.visualstudio.com/api/references/vscode-api>

## Usage Notes

- Use this URL as the primary source for API interfaces, events, classes, and option contracts.
- Verify API names and signatures before implementing or editing extension code.
- Prefer stable APIs unless the task explicitly requires proposed APIs.
- When reviewing code, cross-check command registration, providers, contribution points, and activation behavior against this reference.
