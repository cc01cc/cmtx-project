# Plan Mode Skill

Enforce metadata standards for design documentation. File naming is auto-managed by kilocode.

## Quick Start

### 1. Activate Plan Mode

Open in your editor:

```
.kilo/skills/plan-mode/SKILL.md
```

This Kilo skill is now active.

### 2. Create a Plan

File names are auto-generated by kilocode in `.kilo/plans/`:

```
.kilo/plans/1776264273233-neon-garden.md
```

### 3. Add Frontmatter

Every plan must start with:

```yaml
---
title: "Your Plan Title"
date: 2026-04-15T21:51:16+08:00
---
```

### 4. Validate

Check metadata compliance:

```bash
python3 scripts/validate-plan.py ../.kilo/plans/1776264273233-neon-garden.md
```

Or from project root:

```bash
python3 .kilo/skills/plan-mode/scripts/validate-plan.py \
  .kilo/plans/1776264273233-neon-garden.md
```

## Files

- **SKILL.md** - Main skill documentation and workflow
- **scripts/validate-plan.py** - Automatic validation script
- **references/NAMING_CONVENTIONS.md** - Detailed naming rules

## Rules

### File Location

```
.kilo/plans/
```

File names are auto-generated by kilocode. Examples:

- `1776264273233-neon-garden.md`
- `1776264273234-arctic-wolf.md`

### Metadata Requirements

```yaml
title: "Your Title" (required, string)
date: "2026-04-15T21:51:16+08:00" (required, ISO 8601 + timezone)
```

## Validation

All checks:

1. ✅ File in `.kilo/plans/` directory
2. ✅ YAML frontmatter present
3. ✅ Required fields: `title`, `date`
4. ✅ Date in ISO 8601 format with timezone

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Missing frontmatter | Add `---` block at top with title and date |
| Date format wrong | Use `2026-04-15T21:51:16+08:00` format |
| File not found | Place in `.kilo/plans/` directory |

## Usage Examples

### Create and Validate

```bash
# Validate metadata (file created by kilocode)
python3 .kilo/skills/plan-mode/scripts/validate-plan.py \
  .kilo/plans/1776264273233-neon-garden.md

# Output:
# ✅ File location valid (.kilo/plans/)
# ✅ YAML frontmatter valid (title, date present)
# ✅ Plan file is VALID
```

### Batch Validate

```bash
# Validate all plans
for file in .kilo/plans/*.md; do
  echo "Checking: $file"
  python3 .kilo/skills/plan-mode/scripts/validate-plan.py "$file"
done
```
